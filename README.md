# CPS Orders Mobile

Native Expo wrapper for the Convention Photography Services Order Management System.

## Features
- Native UI built with `react-native-paper`.
- Direct API integration with `https://convention.photos`.
- Android and iOS support.
- EAS Build configuration ready.

## Quick Start
1. Install dependencies: `npm install`
2. Start development: `npx expo start`
3. Build for stores: `npx eas build --platform all`
